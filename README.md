# 🚀 ResearchPilot AI

> An Agentic Multi-Document Research Assistant inspired by Google NotebookLM, built for the Google Build with AI Workshop.

## 📖 Overview

ResearchPilot AI is an AI-powered research companion that helps users analyze multiple documents, discover insights, identify research gaps, generate literature reviews, and compare research papers using Google's Gemini AI.

The application follows an Agentic AI workflow where multiple virtual agents collaborate to process information and generate intelligent outputs.

---

## ✨ Features

### 📂 Multi-Document Analysis
- Upload multiple PDF and TXT documents
- View document previews
- Analyze information across multiple sources

### 🤖 AI-Powered Research Chat
- Ask questions about uploaded documents
- Context-aware responses
- Multi-document reasoning

### 🧠 Agentic Workflow System
The application simulates four AI agents:

- **Planner Agent** – Understands the research objective
- **Retriever Agent** – Collects relevant information
- **Analyzer Agent** – Extracts insights and patterns
- **Reporter Agent** – Generates final outputs

Real-time agent status visualization provides transparency into the research process.

### 🔍 Research Gap Detection
Identify:
- Missing research areas
- Unanswered questions
- Contradictions
- Future research opportunities
- Limitations in existing work

### 📄 Literature Review Generator
Automatically generates:
- Abstract
- Introduction
- Themes
- Findings
- Methodology Comparison
- Research Gaps
- Conclusion

### ⚖️ Document Comparison
Compare multiple documents based on:
- Similarities
- Differences
- Strengths
- Weaknesses
- Methodologies
- Conclusions

### 📚 Source Tracking
- Source-aware responses
- Document references
- Evidence-based research outputs

### 📤 Export Reports
Export generated research outputs and findings for further use.

### 📊 Research Dashboard
Track:
- Uploaded documents
- Questions asked
- Insights generated

## 🔑 Gemini API Setup

1. Visit Google AI Studio
2. Generate a Gemini API Key
3. Launch ResearchPilot AI
4. Enter your API Key in the provided field
5. Start researching

**Note:** API keys are stored locally in your browser and are never uploaded to any server.
